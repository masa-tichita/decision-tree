# fair-oct
## 概要
- 公平性を考慮した最適分類木モデルを混合整数最適化問題として定式化し、解くためのプログラム

## 使用技術
- Python
- poetry

## Installation
仮想環境を作成
```shell
poetry install
```
仮想環境の確認
```shell
poetry env info
```
仮に仮想環境のPythonのバージョンが異なる場合は、以下のコマンドを実行してバージョンを変更する。
```shell
poetry env use python3.11
```
Poe the Poetを有効にする
```shell
poetry run poe
```
